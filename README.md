## How It Works

1. Upload a photo or video (demo version)
2. Click "Share Story"
3. Receive a personalized compliment in Hindlish
4. Like, share, or close your compliment

## Technology Stack

- **Frontend**: HTML, CSS, JavaScript (Static site)
- **AI**: Google Gemini AI
- **Deployment**: Vercel

## Demo

Try the live demo: [StoryCompliment AI on Vercel](https://story-compliment-ai.vercel.app)

## Development

This is a static site that can be deployed directly to Vercel. No build process is required.

To view locally:
1. Open `public/index.html` in your browser
2. Or serve the `public` directory with any static server
